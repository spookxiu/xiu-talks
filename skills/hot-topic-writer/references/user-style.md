# 用户风格参考 - 天宇

## 语言特点

### 口语化表达
- "搁前几年" 而不是 "放在前几年"
- "那阵子" 而不是 "那段时间"
- "干一下子" 而不是 "做这件事"
- "算不明白" 而不是 "无法理解"
- "还早呢" 而不是 "还为时尚早"

### 节奏感
- 短句收尾："一个字'累'"
- 自问自答："那延迟退休对我算什么呢？绝不是..."
- 戛然而止："退休？还早呢。"

### 带态度的描述
- "看着专家吵来吵去"（有主观视角）
- "谁知道裁员和退休哪个先来"（无奈但真实）
- "这也算本事了"（自嘲式认可）

## 结构偏好

1. **开头直接**：年龄/身份 + 核心观点，不绕弯子
2. **中段讲故事**：用具体场景带出感受，不是先给结论
3. **结尾留有余地**：不强行升华，允许"可能吧"这样的不确定

## 避免的表达

| 不要 | 要 |
|------|-----|
| "从某种意义上说" | 直接说 |
| "这代年轻人" | "我"或具体描述 |
| "不难发现" | 让读者自己发现 |
| "让我们..." | 不说 |
| 排比三件套 | 两个或四个，打破工整 |

## 示例片段

**原文风格**：
> 搁前几年，简直是噩梦。那阵子带团队，天天想着怎么让大家能力提升、钱包变鼓、做出来的东西有人用。结果呢？能力有高低，时间不够用，压力从上面来，从客户来，从系统bug来。想做的事做不好，想护的人护不住。一个字"累"。

**关键**：先给场景，再给感受，最后一句定性。
