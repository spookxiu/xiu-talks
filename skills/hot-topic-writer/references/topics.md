# 热点话题获取指南

## 方法

1. **网络搜索**：使用 Tavily 搜索获取今日热点话题
   
   运行命令：
   ```bash
   python3 ~/.openclaw/workspace/skills/openclaw-tavily-search/scripts/tavily_search.py \
     --query "今日热点新闻 社交媒体热议 2025" \
     --max-results 5 \
     --format md
   ```
   
   或搜索其他关键词如："最近热门话题"、"微博热搜"、"抖音热门"等

2. **选择标准**：
   - 有具体事件/人物/地点（不是纯概念讨论）
   - 有争议性或反差感
   - 普通人能有切身感受

3. **呈现给用户**：用1-2句话概括话题，询问用户观点

## 示例话题

- "沈阳12元红烧肉盒饭走红，有人专门坐高铁去吃"
- "年轻人开始去菜市场旅游，早起买新鲜食材"
- "职场'安静辞职'现象：不辞职但不再拼命"
- "县城文学走红：年轻人开始怀念小地方"

## 询问话术

> "最近有个话题：[一句话概括]。你怎么看？用大白话跟我说说你的想法就行。"
